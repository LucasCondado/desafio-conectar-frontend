# Desafio - Sistema de Cadastro e Autenticação de Usuários

Este projeto é composto por um backend (NestJS + PostgreSQL) e um frontend (React) para cadastro, autenticação e gerenciamento de usuários.

## Pré-requisitos

- Node.js (v18+ recomendado)
- npm (v9+)
- PostgreSQL (v12+)
